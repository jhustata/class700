Open#

The advanced class has focused on the process and workflow leading to open science whereas the basic class has been completely dedicated to creating output and content.

Once all output is created and then pushed onto gh-pages, issues of aesthetics will innevitably arise. And markdown, LaTeX, CSS, etc. are very simple languages/tools that help in this regard.

You may read about them on wikipedia. In this class only an introduction to them should set you up for life!
